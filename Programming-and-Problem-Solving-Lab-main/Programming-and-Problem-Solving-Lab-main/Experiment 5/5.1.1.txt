Problem Statement:-Write a Python program that prompts the user to enter a year. The program should determine if the year is a leap year or not and print the appropriate message.



year = int(input())

if (year % 4 == 0 and year % 100 !=0) or (year % 400 == 0):
	print("Leap year")
else:
	print("Not a leap year")
